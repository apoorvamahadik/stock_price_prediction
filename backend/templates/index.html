This API provides stock price predictions using LSTM and ARIMA models.

GET /api/health

Check API health status

POST /api/predict

Get stock price predictions

Request Body:

{
  "ticker": "AMZN",
  "days": 30,
  "model": "lstm"
}
GET /api/stats/<ticker>

Get stock information and statistics

Example: /api/stats/AAPL

GET /api/historical/<ticker>?days=365

Get historical stock data

GET /api/models

Get available prediction models

Frontend Application

The main application is available at http://localhost:3000

Usage Examples

Python:

import requests

response = requests.post('http://localhost:5000/api/predict', 
    json={
        'ticker': 'GOOGL',
        'days': 30,
        'model': 'lstm'
    })
data = response.json()

JavaScript:

fetch('http://localhost:5000/api/predict', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        ticker: 'MSFT',
        days: 30,
        model: 'arima'
    })
})
.then(response => response.json())
.then(data => console.log(data));
